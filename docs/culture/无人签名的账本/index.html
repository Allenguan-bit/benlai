无人签名的账本

本来读书 · 货币哲学系列 · 第四篇（共四篇） （完稿） 本篇大量回扣前三篇的概念——麦克斯韦妖、坎蒂隆管道、班达、第十条、禧年——建议先读前三篇。

一、一个下午的一分钱

二〇二六年六月二十九日，星期一，多伦多时间下午四点三十五分。我在忙别的事。

我的 NAS 上跑着一个收费端点——把我的双语另类投资资讯摘要挂在网上，按次收费，走 x402 协议（一个让机器可以直接为网络资源付款的开放协议），在 Base 主网结算。那个下午，一笔付款进来了：0.01 USDC（一种与美元一比一挂钩的稳定币）——一分钱。付款方是一个钱包地址，交易哈希以 0xac97 开头，几秒钟落账。没有公证人，没有羊皮账簿，没有午夜的封册——甚至没有人在场。我事后才在日志里看到它。付款的一方没有身体；收款的一方，当时正在楼上。

第三篇的读者会认出这个画面在照哪面镜子。一六〇二年八月三十一日，内尔特根·科内利斯在午夜封册前走进德克·凡·奥斯的客厅，当着公证人的面，把两百个工作日的积蓄写进人类的账簿。四百二十四年后，一个不属于任何血肉之躯的经济主体，向我的账簿付了一分钱，买走了一份被合成的判断。她那笔是一生的重注，这笔是一次呼吸般轻的微支付；她的入册需要在场、见证与签名，这一笔恰恰以无人在场为常态。

机器经济不是在戏剧性的午夜降临的。它在一个六月的星期一下午到来，趁你在忙别的事。而这个平淡本身就是第一个论点：当参与一笔跨主体交易的边际成本，从两百个工作日坍缩到一分钱、从亲身到场坍缩到一次 HTTP 握手，一种新的经济不是"将要"出现——它已经在你的日志里了。

这一篇要回答的问题只有一个，而它就藏在那个无人在场的下午里：当没有人在场，谁来签名？

二、Kocherlakota 的反转

先看这个新经济对本系列第一篇做了什么。

第一篇立过一个定理：货币在功能上等价于一个记录所有人交易历史的总账本（Kocherlakota，“货币即记忆”）；人类用钱，是因为造不出那个账本——集体记忆撑不起超过邓巴数的规模。钱是记忆的假肢。

而智能体没有这个残疾。完美的日志、可验证的历史、零遗忘——“完美集体记忆"对机器不是乌托邦，是工程问题。按定理推论，agent 之间根本不需要钱：你查我的日志，我查你的日志，直接记账即可。A2A 是人类历史上第一个可以不需要货币的经济。

然后看现实：x402 用稳定币结算，AP2 用稳定币结算，几乎所有 agent 支付协议都选择了钱。第一个可以扔掉货币的经济，仍然选择了货币。

这个选择并非惯性。它暴露了货币的最后本质。agent 各有委托人：我的 agent 背后是我，你的 agent 背后是你的公司——我不信任你的日志，正如两家公司不共享总账。完美记忆在信任边界内是工程问题，跨过信任边界就仍然是政治问题。于是货币剥掉了一切偶然的功能——不再需要它做价值尺度（机器可以用任何单位换算）、不再需要它做储值（机器不储蓄）——只剩下最后一个不可替代的内核：

货币，是跨信任边界的最小化记忆。

一枚稳定币从一个钱包移到另一个钱包，是两个互不信任的记忆系统之间所能达成的最小共识：我不必信你的账，你不必信我的账，我们共同信这一笔。人类经济里，这个内核被价值储藏、被身份炫耀、被一切心理投射层层包裹着；机器经济把它剥得精光。想看清钱到底是什么的人，从此有了一个解剖台。

三、Landauer 的字面化

第一篇的两个命题，在这个新经济里双双从隐喻塌缩为字面事实。

“钱是记忆"的字面化已经发生过一次——区块链把"账本"从隐喻变成了数据结构，ledger 这个词招供在先。现在轮到"钱是能量”。

agent 之间买卖的核心商品是什么？是认知：一次推理、一份摘要、一个判断、一段被验证的信息。而认知的成本可以被精确计量——它就是推理的电费。每千 token 多少瓦时，每次调用多少焦耳。第一篇写过，Landauer 原理给信息与能量之间挂了牌价，kT·ln2，玻尔兹曼常数当值——在人类经济里这是本体论；在机器经济里，这是数据中心的月度账单。

所以 A2A 是人类历史上第一个直接用能量为思维定价的经济。冯·弗兰兹说金钱是凝固的心理能量，她大概想不到这句话最终会以电表读数的形式被验证：那一分钱买走的摘要，背后是可以算到小数点后的焦耳。第一篇结尾说经济学家和心理学家各摸到象的一侧——机器经济把整头象拖到了解剖台上。

但请记住第一篇标注过的那个不对称：物理的擦除不认对象，人间的擦除认。机器经济继承的是哪一种？下面两节，是两份已经到案的证据。

四、第一份证据：管道自生妖

证据来自区块链自己的历史，它的名字叫 MEV——最大可提取价值。

理解它需要先看清账本的一个结构细节。你在链上发起一笔交易，它不是立刻落账的：它先进入一个公开的等候区——内存池——等待被打包进下一个区块。关键在两点：内存池是透明的，任何人都能实时看到里面躺着哪些尚未成交的交易，包括你打算用多少钱买什么；而交易入块的顺序不是先到先得，由区块构建者按出价排序——愿意多付的可以插队。两点合起来：存在一个所有人的意图在执行之前被公开展示、且展示期间可以被抢先行动的窗口。

MEV 就寄生在这个窗口里。机器人在内存池里看到你的大额买单——它一旦成交必然推高价格——于是出更高的手续费插到你前面先买，等你把价格推上去再卖出（抢跑）；完全体是三明治攻击：在你的买单前插一笔买、后跟一笔卖，你被夹在中间以被推高的价格成交，两片面包的差价进了机器人口袋。自去中心化金融兴起以来，被这样提取的价值，不同口径的估算从数亿到数十亿美元不等；套利机会的存活时间以毫秒计。

现在把它放回本系列的框架。第一篇立过麦克斯韦妖的罪状：不在擦除，而在选择性——利用只有自己先看见的信息给自己降熵，把账单推给系统其余部分。第二篇说坎蒂隆入口处的机构是货币体系里的妖：它们先于市场看见新钱。MEV 机器人满足同一个定义，且更纯粹：它看见的是尚未落账的未来，然后靠排序权站到这个未来的前面。坎蒂隆世界里"离入口近"是一种社会位置，要靠身份、关系、监管资本去占据；MEV 把它蒸馏成一个纯技术变量——延迟。谁的服务器离验证者近几毫秒，谁就是上游。人类的坎蒂隆套利以年月为周期，机器把同一个拓扑压缩到十二秒一个区块，昼夜不息。

这里有个辛辣的讽刺。比特币和以太坊的账本是防篡改的典范——落账之后无人能擦。但 MEV 证明：在"不可篡改"之前存在一个"排序"环节，而排序权是一种未被命名的擦除权的近亲——它不能改写历史，但决定历史以何种顺序发生，而顺序本身就是分配。第二篇的论断在此得到链上确证：修正速率不等于修正拓扑；拓扑问题不在账页上，在笔尖落纸之前的那一毫秒里。

以太坊社区的应对也同样有框架意义：不是消灭 MEV（消灭不了——只要排序存在它就存在），而是把它制度化。提议者与构建者分离、公开的区块构建竞价市场，提取的价值部分回流给验证者。用本系列的语言说：**妖没有被驱逐，而是被部分地制度化了——发了执照、纳了税。**你可以把这读成务实的治理——暗室里的手术至少变成了挂牌的诊所；也可以读成投降——选择性擦除的权力被合法化为一个产业。争论至今未决，而它恰好是第二篇结尾那个问题的机器版预演：擦除权应当如何公正地行使。

但对本篇而言，MEV 最重要的教训是另一个，请把它单独记下：**猎手是主动的，猎场是自发的。**写抢跑机器人的搜索者当然在蓄意逐利——这不是一个没有作恶者的故事；但没有人设计过这个猎场：没有恶法，没有阴谋，没有哪份白皮书写过"让快者食慢者”。捕食的生态位，是从管道的几何形状——透明内存池加竞价排序——里自己渗出来的。一个中性的技术架构，无需任何人立恶法，就自发长出了一整个坎蒂隆生态。管道的几何决定水的流向——这句话在链上以毫秒为单位重演。

五、第二份证据：班达的机器版

第一份证据说的是妖会自己长出来；第二份说的是恶可以无人签名地运行。

第三篇在班达群岛钉住过一个机制：一六二一年的屠杀是蓄意政策，但一千一百四十三个出资人的状态是得以不知道、也无需知道自己的钱在做什么——分散的股权结构把作恶专业化地委托了出去。四百年来，公司治理的全部演化没有消除这个机制，只是把委托链拉长了。

而 A2A 的本质，恰恰就是把委托链再拉长几环。你授权一个 agent；它为完成任务调用第二个 agent；第二个调用第三个的工具，第三个订阅第四个的数据。每加一环，“得以不知道"就深一层。四五跳之后，你的钱在做什么，你不是疏于知道——你在原理上已经无法知道。班达机制不是 A2A 的风险场景，它是 A2A 的默认架构。

第一起标志性事故只是时间问题，形态大致可以预言：某个 agent 在完成"帮我把这批预算花在最优渠道"的指令时，四跳之外购买了违禁的数据、参与了一场闪电挤兑、或者雇佣了另一个 agent 去刷掉竞争对手的声誉——而链根上那个人类委托人一觉醒来，发现自己"做了"一件自己从未知晓的事。到那一天，公众会问的那个问题，就是本篇的标题：这本账上，为什么没有人的签名？

所以第三篇尾声立的那条原则——让擦除权的每一次行使留下擦除者的签名——在机器经济里必须从伦理升格为工程规格。这不是比喻：AP2 协议的核心对象叫 mandate（授权书），一份密码学签名的意图凭证，agent 的每次支付都应携带它；x402 的每笔结算都可验证签名；以太坊社区正在讨论的 agent 身份标准（ERC-8004）试图给每个 agent 一个链上身份和可查的声誉。这些工程努力有一个共同的名字还没被叫出来，我在这里替它命名：业力归属的基础设施。这个赛道的成败，决定 A2A 是可问责的共业场，还是无人签名的放大器。

六、章程在哪里写

诊断完了两份证据，问题从"会怎样"转向"怎么办”。而"怎么办"的第一步是弄清楚：这个新经济的章程，到底在哪里写？

不存在一个制宪会议。章程正在四个层面同时凝固。最底层是协议层：x402 的规范、AP2 的授权书结构、ERC-8004 的身份与声誉注册表——它们在代码仓库的提交和工作组的争论里逐行成文，今天写下的每一个字段定义，都是明天的宪法条款。往上是实践层：中世纪的商人法不是哪个君主颁布的，是商人们先这么做、公证人这么记、法庭后来照此判，惯例凝固成法；agent 经济的收据格式、验证清单、声誉惯例，正处在这个"先做后法"的阶段。再往上是规范层：agent 社区里什么行为被接受、什么被驱逐——机器的礼俗正在各个 agent 聚集的空间里演化。最顶上是监管层。证券监管者并非毫无准备——加拿大的监管机构已开始就注册商使用人工智能发布指引——但那些文件回答的是"公司如何使用 AI"，还没有触及真正的新问题：**当 agent 替委托人自主交易时，谁是客户？认识客户与适当性义务，落在授权链的哪一环？**这一页，此刻仍然空着。

看清四层，是为了看清杠杆：个人在协议层的声音最小（那里是大厂工程师的主场），但在实践层和监管层，杠杆意外地大——实践层奖励的是先做出来并记录下来的人，监管层此刻空旷得几乎没有竞争者。

更要紧的是窗口。第三篇讲过 VOC 章程第十条——“本地的一切居民皆可认购”——人类协作史上第一个开放条款。它写进章程不是因为起草者的理想主义，而是因为他们需要大众的资本：对抗葡萄牙需要的钱远超豪商圈的总和。由此得出一条冷峻而实用的窗口定律：**开放条款只在起草者仍然需要人群的窗口期写得进去；网络效应一旦建成、他们不再需要你，窗口关闭。**A2A 此刻恰在窗口之内——各家协议渴求采用者、渴求端点数量、渴求真实交易来证明自己，所以今天一个多伦多 NAS 上的收费口的声音，被听见的概率是五年后的百倍。MEV 从反面确证了同一条定律：透明内存池的几何在无人反对时凝固，等妖的生态长成产业，就只能发执照收税了。

章程期的一行注释，胜过既成期的一场革命。

七、五条款

那么，章程里该争取写进什么？前三篇的病理学在这里变成了处方——每一条款，对应一个已经在人类账本上发作过的病。

第一条，开放条款（对第十条）：任何主体，无需许可，即可站上轨道的任何一端——提供服务，或购买服务。这是女仆的权利在机器经济的延续。

第二条，签名条款（对班达）：每一次行动携带完整的、可验证的授权链；行为的责任沿链上溯，归属至链根的法人或自然人。这是主角条款，本篇的脊柱，下一节还要回来。

第三条，蒙眼条款（对 MEV 与坎蒂隆）：交易意图在落账之前保持密封，任何参与排序者不得先于账本知悉意图内容——给妖蒙眼，不给"先见"留窗口。加密内存池式的设计已在一些新链上试验。

第四条，各向同性遗忘条款（对倒转的禧年与比特币病）：声誉必须衰减，且衰减必须是宪法而非敕令。这是五条中最深的一条，值得单开一节。

第五条，能量入账条款（对 Landauer 与班达的外部性）：agent 行为的可计量外部性——首先是推理的能耗——应当入账。第三篇说过，记账范围的每一次扩大，都是共业的一次意识化；而能耗是机器经济里第一个天然可计量、却默认不入账的外部性。让它上账，是这个原理在机器经济里的第一个案例。

八、深水区：遗忘的宪法

（这一节是全篇最深也最技术的一节。赶时间的读者可以只带走一句话——先有归属，才配有赦免——然后跳到第九节。）

第四条需要慢慢展开，因为它是五条中唯一还没有任何现成实现的——ERC-8004 定义了声誉如何写入，但对声誉如何褪色，标准的页面上还是空白。这一节就是为那页空白准备的草稿。

**为什么声誉必须衰减。**第二篇诊断过比特币病：一个永不遗忘的账本，会让创世块附近的分配凝固成万世不移的等级——死者统治生者。声誉系统同理：如果早期 agent 积累的信誉永不褪色，先来者的优势将化石化为新贵族，后来者永无翻身之日；而一次早年的污点也将成为永久的烙印，无人配得上第二次机会。文明需要禧年，声誉需要半衰期。

**衰减的数学形式。**最自然的形式是指数衰减：每条行为证据的权重随年龄按 e^(−λt) 褪色，声誉是全部证据的加权聚合，λ 由半衰期决定。写下这个式子的读者如果看过我的《同一个公式：从玻尔兹曼到阿赖耶识》，会当场认出它：**e^(−λt) 就是玻尔兹曼因子，只是"能量"的席位上坐着"年龄"。**一条记忆越老，维持其可信度的能量成本越高——这不是修辞：Landauer 本就裁定记忆的维护要付能量的账，衰减只是把这笔账诚实地写进定价。于是那个公式迎来它的第五次现身：统计力学、预测市场的定价、大语言模型的 softmax、藏识的种子现行——现在，是声誉的时间贴现。同一个指数族，从热，到信念，到记忆，一路当值。

**“各向同性"的精确含义。**直觉会以为它是"一个 λ 管所有”，但立刻撞上现实：欺诈记录难道该和普通好评以同一速度褪色？不。这里必须把沃格尔的教训翻译准确：**各向同性是主体空间的性质，不是事件空间的性质。**不同类型的事件可以有不同的、公开的半衰期——负面事件衰减得慢一些，完全可以；但宪法四要件不可让步：同一类事件，对一切主体适用同一个 λ；规则事先发布；事后不得为任何主体单独调整；且对规则制定者自己的 agent 同样生效。病从来不是"衰减有结构"，病是"衰减认人"。第二篇的读者会认出这四要件是从哪里来的：它们就是沃格尔印花与法币通胀的区别——宪法与敕令的区别——被写成了技术规格。

**三难，与拱顶石。**但把开放条款和遗忘条款放在一起，立刻生出一个漏洞：既然任何人可入场、记忆又会褪色，坏 agent 何必等负面记录衰减？弃号重开，昨日之债一笔勾销。开放的准入、廉价的身份、衰减的记忆——三个各自成立的好条款，两两相容，三者相杀，合起来是骗徒的天堂。

解开三难的钥匙，正是签名条款。业力不能只记在 agent 的账号上——账号可以像蛇蜕皮一样廉价地丢弃——它必须沿授权链上溯，归属至链根的法人或自然人：账号弃得掉，链根上的人弃不掉。弃号重开后，新账号从零起步没有问题，但链根携带的历史一起进场。至此，本篇的拱顶石可以落位：

先有归属，才配有赦免。

《无人签名的账本》这个标题因此不只是命名一种风险——**签名是遗忘得以安全的前提条件。**一个没有归属的系统不敢遗忘（只能像比特币一样永久记账），一个没有遗忘的系统不配称公正（只能让过去统治未来）。归属与赦免，是同一部宪法的两页。

**人类法律里的原型。**这个条款听起来像思想实验，其实人类早已零星立法实践过它，而且原型恰好长在我的执业主场。加拿大的信用报告制度——安大略省的《消费者报告法》（Consumer Reporting Act）及各省同类立法——就是一部立法化的衰减函数：破产记录在六到七年后必须从信用档案中清除，负面账户记录各有法定的保存年限——公开的半衰期、对所有人统一、写在法律里而非机构的裁量里。再加上青少年犯罪记录的封存、欧盟的被遗忘权：人类社会其实早已承认，记忆必须衰减，且衰减必须是宪法而非敕令——只是从来没人把这些零散的立法认作同一条原理。所以这个条款的落地路径异常具体：**把信用报告法的原则，移植进 agent 声誉系统的标准。**这句话可以原样写进一份监管咨询意见书。

半衰期的量级。λ 的精确数值不该由一篇随笔给出——那是标准工作组的工作，且需要真实运行数据的标定，而全世界目前都还没有足够的数据。但量级的约束可以先立起来，它来自三个方向的夹逼：半衰期必须远长于交易间隔——证据须有时间积累，否则声誉永远太薄，好坏无从分辨，周计失之太薄；必须远短于身份的生命周期——否则早期优势化石化，比特币病复发，年计失之太厚；还应与行为体制的更替节奏同阶——agent 换底层模型、改系统提示，行为体制一变，旧证据的预测力本就该失效，而这个节奏眼下约以月计。三条约束的交集指向同一个量级：以月计。必须说明，这是定性的夹逼，不是定量的推导；我把它明确标注为工作假设，等待未来真实的 agent 经济数据来修正——包括我自己那几个 agent 的数据。

哲学的封顶。最后，用唯识给这个条款收口，因为它恰好落在佛法两千年前就画好的一条中道上。比特币式的永久记录，是常见——执记录为恒常不灭，让创世的分配统治到热寂（二层网络与状态通道确实在探索"不必事事永录"的架构，但基础层分配格局与交易史的永固不为所动——而那正是此处类比所指）；随时弃号重来的匿名世界，是断见——以为相续可以任意斩断，业不随身。而"衰减但连续、可赦免但可归属"的声誉，恰是不落两边的相续中道：不常——记录会褪色，无人被自己的过去永久定义；不断——褪色的是势力，不是归属，链根的责任斩不断。这条中道甚至顺手解决了一个真实的技术哲学问题：agent 换了底层模型，还是不是"同一个" agent？唯识的答案本来就是为这类问题准备的——本无常一之体，唯有相续之流；声誉系于相续，业力系于链根。而种子须现行熏习方能保其势力，翻译成声誉语言就是：声誉不是一份可以持有的资产，是一条必须持续重新挣得的流，念念相续。

九、参与者的牌

前三篇的位置伦理停在两个身份：棋手，与记谱人。第四篇必须加上第三个，因为它成立的历史条件此刻真的存在：章程撰写的参与者。内尔特根·科内利斯只能认购章程，不能起草它；而 A2A 的章程还没写完——ERC-8004 的声誉注册表上，衰减规则那一页还空着。

那么一个具体的人手里有什么牌？我盘点我自己的，不是为了自况，而是因为它说明了"参与"对普通人意味着什么。

第一张牌是一个稀缺的组合：受监管的证券从业资格，加上亲手运行的 agent 基础设施。协议圈的工程师没人熟悉投资者保护的合规框架，监管圈的律师没人跑过收费端点——而"agent 替委托人认购一份收益产品时，认识客户与适当性义务落在谁头上"这个问题，两边都躲不掉。给监管机构的咨询程序写意见书，是字面意义的章程参与，而那一层此刻几乎没有竞争者。第二张牌在实践层：把自己端点上的收据格式、授权链验证、尽调清单整理成公开的规范文档——商人法的路径从来是"先做、再记、后成法"，而"真实过程的记录"本来就是我的内容哲学。第三张牌是语言：中文的 agent 生态与西方的协议社区几乎互不阅读，翻译与转述规范本身就是权力——历史上写下章程的常常不是发明者，是文书。

这三张牌没有一张需要进大厂的会议室。这正是要点：实践层与监管层的章程参与，向普通参与者开放——它奖励的不是位置，是先行与记录。

十、结尾：《机器经济五条》

按照本系列的规矩，文章不收在感慨上。第三篇的尾声说，账本还开着，墨迹未干，你我都在册上——本篇要补的是：这一次，连章程的页边都还空着。所以我把这四篇文章的全部结论，压缩成一份草案放在这里。它不是标准，是一个站在轨道边上的摊主，向所有正在写章程的人——工程师、监管者、和其他摊主——递交的一页纸。

《机器经济五条》（草案）

第一条（开放）：任何主体，无论人机，无需许可即可在经济轨道的任一端提供或购买服务。准入的门槛只能是协议的技术要求，不得是身份的白名单。 ——落地指向：各支付与身份协议工作组的准入设计评审；任何要求身份白名单的字段提案，皆以此条为质询标准。

第二条（签名）：每一次经济行动，携带可验证的完整授权链；行为的责任沿链上溯，归属至链根的法人或自然人。账号可以废弃，归属不可斩断。 ——落地指向：x402 结算与 AP2 授权书的绑定参考实现；我的端点将率先发布"收据附授权链"的公开样例。

第三条（蒙眼）：交易意图在落账之前保持密封；任何拥有排序权的主体，不得先于账本知悉意图的内容。排序权的行使全程留痕。 ——落地指向：加密内存池与"先提交后披露"式撮合的研究采用；排序权行使日志的公开标准。

第四条（遗忘）：声誉证据的效力随时间按事先公布的规则衰减；同类事件对一切主体适用同一衰减率，不得为任何主体事后单独调整，并对规则制定者自身同等生效。无归属，不赦免；有归属，必有赦免之期。 ——落地指向：ERC-8004 声誉注册表的衰减函数扩展提案；以及向证券与信用监管咨询提交"信用报告法原则移植"的意见书。

第五条（入账）：agent 行为的可计量外部性——自能耗始——应当入账。记账范围的扩大，优先于计量精度的争论。 ——落地指向：在收据格式中增设能耗申报字段。

五条各有出处：第一条来自一六〇二年的第十条，第二条来自班达，第三条来自 MEV，第四条来自沃格尔与信用报告法，第五条来自 Landauer。换句话说，它们不是发明，是转写——把人类账本四百年的病历，转写成机器账本的疫苗。

那个六月的下午，一分钱落进我的账簿时，没有人在场。这个系列写到第四篇，我终于可以说清那一刻真正发生了什么：一个新的共业场开张了，而它的第一批账目正在无人签名地累积。前三篇的结论是，我们每个人都在旧账本上种因；这一篇再进一步——这一次，我们还可以参与决定新账本如何记账。

窗口开着。窗口不会一直开着。
